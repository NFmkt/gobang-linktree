# 고방 링크트리 — 확정 스펙 (v1)

> 인스타그램 프로필용 자체 멀티링크 페이지 + 클릭/유입 통계 + 관리자.
> `@grill-me` 인터뷰로 확정. 코딩은 2단계(디자인 시스템) 확정 후 시작.

## 1. 목적
- 인스타 프로필(@gobang.kr) 링크용 자체 Linktree 대체 서비스.
- 링크를 유연하게 관리하고, 클릭·유입 데이터를 **직접 축적·분석**.

## 2. 아키텍처 / 스택
| 항목 | 결정 |
|---|---|
| 관리 방식 | 자체 **관리자 페이지 + DB** |
| 프레임워크 | **Next.js (App Router)** — 공개 페이지 + 관리자 한 몸 |
| DB / 통계저장 | **Supabase (Postgres)** |
| 배포 | **Vercel** (기본 도메인 `xxx.vercel.app`, 커스텀 도메인 없음) |
| 관리자 인증 | **단일 비밀번호(환경변수)** — `/admin` 접속 시 비번 1개 |
| 로컬 포트 | **3939** (`시작 3939.bat` 생성) |
| 작업 위치 | `my-claude/for_Release/gobang-linktree/` |
| 언어 | 한국어 전용 |
| 반응형 | **모바일 우선**, 데스크톱 대응 |

## 4. 링크 항목 데이터 모델
| 필드 | 필수 | 설명 |
|---|---|---|
| title | ✅ | 버튼 텍스트 |
| url | ✅ | 이동 주소 |
| icon | ✅ | 사전제작 **커스텀 SVG 아이콘 세트에서 드롭다운 선택** (icon-design) |
| subtitle | | 짧은 부연 |
| order | ✅ | 드래그로 순서변경 |
| active | ✅ | 노출 on/off (임시 숨김, 운영 테스트용) |
| thumbnail | | (선택) 카드형 이미지 |

- 관리자에서 링크 **추가 / 수정 / 삭제 / 드래그 순서변경** 가능 (핵심 요구).

## 5. 제휴·협력 문의
- **`mailto:{저장된주소}`** 링크 버튼 방식 (서버 발송/폼 없음).
- 기본 수신 주소 `neoflatworks2@gmail.com`, **관리자에서 실시간 수정 가능**.
- 제휴 버튼 클릭수는 통계로 집계.

## 6. 통계(Analytics)
- **측정 이벤트**: 페이지뷰 / 링크별 클릭 / 유입출처.
- **유입출처**: HTTP Referrer + UTM 파라미터 (인스타 링크에 `?utm_source=instagram` 태깅 운영법 함께 제공).
- **클릭 기록 방식**: **비콘(`navigator.sendBeacon`)** — 클릭 즉시 이동 + 백그라운드 전송.
- **관리자 대시보드(요약)**: ①총 방문수 ②링크별 클릭수 순위 ③최근 7/30일 추이 ④유입출처 top.
- **보관**: 무기한. 관리자에서 **수동 삭제/초기화** 가능.
- **개인정보**: 익명 집계 — IP 원문 저장 안 함, referrer/utm/링크id만.
